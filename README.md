# Birlik Bahçesi (Garden of Unity) 🌻

This is a digital support and appreciation project created with love for our teacher, **Sunay Dinçer**. The project allows students and well-wishers to send virtual flowers and leave heartfelt notes in a shared digital garden to show their support.

## 🌟 Features

- **Virtual Garden:** A beautiful visual representation of appreciation.
- **Send Flowers:** Click a button to instantly add a virtual flower to the garden.
- **Leave Notes:** Share your thoughts, appreciation, and support messages.
- **Live Statistics:** See the total number of flowers and notes shared in the garden.

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## 🚀 Deployment

This project is ready to be published on **GitHub Pages**. Since it consists purely of static files (HTML, CSS, JS), no build process is required.

**To deploy:**
1. Fork or clone the repository to your GitHub account.
2. Go to your repository's **Settings > Pages**.
3. Under the "Build and deployment" section, choose **Deploy from a branch**.
4. Select your main branch (e.g., `main` or `master`) and keep the folder as `/ (root)`.
5. Click **Save** and wait a few minutes for your site to go live!
